yolo-happiness
==============

Un space shooter basico usando ncurses.

Los controles son 'J' para subir, 'K' para bajar y 'Espacio' para disparar.


Para compilar en Debian hacen falta estos paquetes:
apt-get install ncurses-devel ncurses
